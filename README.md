 ## finonex assignment running instructions




###   Installation


1. Clone the repo
   ```sh
   git clone https://github.com/dsabag/finonex.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. run db.sql in the postgres cli
   
4. create an "events.jsonl" file with relevant data
   
5. run the server
   ```sh
   npm run server
   ```
6. run the client
   ```sh
   npm run client
   ```
7. run the processor
   ```sh
   npm run processor
   ```
